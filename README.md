# JANO Logs Repository

Este repositorio contiene los logs del sistema JANO, actualizados automáticamente por el servicio `jano-healthcheck`.

## Estructura

```
logs/
└── YYYY-MM-DD/
    ├── healthcheck.log
    └── activity.log
```

## Actualización

Los logs se actualizan automáticamente:
- En el arranque del sistema
- Cada 6 horas

## Privacidad

Este repositorio puede contener información sensible. Manténgalo privado.
